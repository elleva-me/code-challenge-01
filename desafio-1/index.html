Exercício 04 (150 pts)

Entrada:

Saída:

Mover os números pares para a frente do vetor

Você deve implementar a função moverNumerosParesParaAFrenteDoVetor no arquivo exercicio-04.js que dado um vetor como entrada, a função deverá retornar um vetor com os números pares na frente e posteriormente os números ímpares. Vale destacar que o número 0 deve ser considerado como número PAR. Vejamos alguns exemplos abaixo:


Entrada:
1,3,4,5,6,9,10,25
Saída:
4,6,10,1,3,5,9,25

Entrada:
3,5,21,17,48,1,55
Saída:
48,3,5,21,17,1,55
